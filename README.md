مشروعي هو موقع ويب يجمع جميع شركات تأجير السيارات في العراق في مكان واحد. الهدف من المشروع هو تسهيل عملية البحث عن السيارات المتاحة للتأجير من مختلف الشركات في العراق ومقارنة الأسعار والخدمات المقدمة.

الواجهة الأمامية:

React: تم استخدام React لإنشاء واجهة المستخدم الديناميكية والقابلة للتفاعل.

Leaflet: تم استخدام Leaflet لعرض الخرائط وتحديد مواقع شركات تأجير السيارات على الخريطة.

Axios: تم استخدام Axios للتواصل مع الواجهة الخلفية لجلب البيانات وإرسال الطلبات.

Tailwind CSS: تم استخدام Tailwind CSS لتنسيق وتصميم الموقع بطريقة حديثة وسهلة.

الواجهة الخلفية:

Firebase: تم استخدام Firebase لإدارة قاعدة البيانات والمصادقة.

Express.js: تم استخدام Express.js لإنشاء API والتحكم في العمليات الخلفية.

Multer: تم استخدام Multer لتحميل الملفات مثل الصور.

Bcrypt: تم استخدام Bcrypt لتشفير كلمات المرور لضمان الأمان.

JWT: تم استخدام JSON Web Token (JWT) لإدارة الجلسات والمصادقة.

الوظائف الرئيسية:

عرض السيارات: يمكن للمستخدمين تصفح السيارات المتاحة للتأجير من مختلف الشركات، وعرض التفاصيل مثل الأسعار والمواصفات.

البحث والتصفية: يمكن للمستخدمين البحث عن السيارات باستخدام فلاتر متعددة مثل نوع السيارة، السعر، والموقع.

التسجيل والدخول: يمكن للشركات والأفراد إنشاء حسابات وتسجيل الدخول لإدارة حجوزاتهم أو عروضهم.

إدارة الإعلانات: يمكن للشركات إضافة وتحديث وحذف إعلانات السيارات الخاصة بهم.

Your project is a website that aggregates all car rental companies in Iraq into one platform. The goal of the project is to simplify the process of searching for available rental cars from various companies in Iraq and comparing prices and services offered.

Frontend:

React: Used for building the dynamic and interactive user interface.

Leaflet: Used for displaying maps and locating car rental companies on the map.

Axios: Used for communication with the backend to fetch data and send requests.

Tailwind CSS: Used for styling and designing the website in a modern and easy way.

Backend:

Firebase: Used for managing the database and authentication.

Express.js: Used for creating the API and handling backend operations.

Multer: Used for file uploads such as images.

Bcrypt: Used for encrypting passwords to ensure security.

JWT: Used for managing sessions and authentication through JSON Web Tokens (JWT).

Main Features:

Car Listings: Users can browse available rental cars from various companies and view details such as prices and specifications.

Search and Filter: Users can search for cars using multiple filters such as car type, price, and location.

Registration and Login: Both companies and individuals can create accounts and log in to manage their bookings or listings.

Ad Management: Companies can add, update, and delete their car rental listings.
